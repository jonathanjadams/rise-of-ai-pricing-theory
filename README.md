# rise of ai pricing theory

This repository contains replication code for the theoretical model and analysis in [_The Rise of AI Pricing: Trends, Driving Forces, and Implications for Firm Performance (2026 JME)_](https://www.sciencedirect.com/science/article/pii/S0304393225001461).

## Instructions:
Clone the repository, and set your Matlab directory to the repository folder.  Then run **ai_pricing_model.m** which will pull from the data folder, and produce figures.  


## Dependencies:
 - Matlab (code was tested on version 2024b)


## Output
- figures/computing_price.png (Figure 5a)
- figures/pricing_adoption.png (Figure 5b)
